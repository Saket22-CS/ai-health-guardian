## ūüĆć About

**AI Health Guardian** is an intelligent health prediction platform built for **UN SDG Goal 3 ‚ÄĒ Good Health and Well-Being**. It combines a Random Forest ML model trained on 4,920 clinical samples with Google's Gemini AI to deliver:

- Instant disease prediction from 131 symptoms across 41 diseases
- Personalised health risk scoring with radar chart visualisation
- Real-time AI health chatbot (Dr. AI) powered by Gemini Flash
- Medicine information lookup, health dashboards, and PDF-ready reports

> *Making quality health information accessible to everyone.*

## ūü§Ė ML Model

```

         ‚Üí  Best Model Auto-Selected
         ‚Üí  Saved as .pkl
```

## ‚öôÔłŹ Troubleshooting

| Problem | Solution |
|---------|----------|
| `python train_model.py` ‚ÄĒ KeyError: 'prognosis' | Your CSV uses 'Disease' as target ‚ÄĒ already handled automatically |
| `ValueError: invalid literal for int()` | Text-format dataset ‚ÄĒ already handled, the script converts it |
| Gemini 404 model not found | Update `FREE_MODELS` list in `gemini_helper.py` |
| Gemini 429 quota exceeded | App auto-retries and falls back to next model; wait 1 min |
| Model not found error in app | Run `python train_model.py` first |
| Port already in use | `streamlit run app.py --server.port 8502` |
| `venv` won't activate on Windows | Run PowerShell as Admin ‚Üí `Set-ExecutionPolicy RemoteSigned` |
